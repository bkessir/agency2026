# 🦌 AIM Pronghorn 404 — Visual Identity Guide

## 1. Brand Overview

**Name:** AIM Pronghorn 404  
**Essence:** Speed • Precision • Intelligence • Clean Tech

### Concept
- **Pronghorn** → fastest land animal (speed + agility)
- **404** → developer culture, problem-solving mindset
- **Meaning** → Fast, precise builders who solve what others can’t

---

## 2. Logo

### Primary Logo
Stylized geometric pronghorn head:
- Symmetrical
- Sharp edges
- Minimal + modern

### Logo Usage Rules
- Maintain clear space (equal to nose height)
- Always preserve proportions
- Use only approved colors

### Do NOT
- Stretch or distort
- Add shadows or effects
- Change colors
- Place on busy backgrounds

---

## 4. Typography

### Headings
- Font: Inter / Space Grotesk / Sora
- Bold, clean, slightly spaced

### Body Text
- Font: Inter / Roboto
- Medium weight, high readability

### Code / Technical
- Font: JetBrains Mono / Fira Code

---

## 5. Layout & UI Style

### Principles
- Minimal
- High whitespace
- Structured grid
- Clear hierarchy

### Sections
- Alternating light/dark
- Smooth transitions

---

## 6. Components

### Buttons
- Rounded (8–12px)
- Gradient or solid
- Hover = glow or shift

```css
background: linear-gradient(135deg, #C47A2C, #8B5CF6);
```

### Cards
- Soft shadow
- Subtle border
- Hover lift

### Inputs
- Rounded
- Thin border
- Focus highlight

---

## 7. Imagery Style

### Direction
- Minimal
- Geometric
- High contrast
- Subtle gradients

### Avoid
- Busy visuals
- Overly realistic images
- Stock-heavy design

---

## 8. Iconography

- Thin stroke
- Rounded edges
- Consistent sizing
- Use Lucide / Heroicons

---

## 9. Motion & Interaction

- Duration: 200–300ms
- Ease-in-out

### Hover
- Scale: 1.02–1.05
- Glow or gradient shift

### Transitions
- Fade + slight movement

## 12. Do / Don’t

### Do
- Keep it clean
- Use strong contrast
- Maintain symmetry
- Focus on usability

### Don’t
- Overdesign
- Add extra colors
- Use inconsistent styles
- Break layout structure
